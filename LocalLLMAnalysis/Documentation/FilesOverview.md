# LocalLLMAnalysis Documentation -- Files Overview

| File                                                        | Description                                                                                                                                                                                                                                                                          |
|-------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Architecture Analysis Toolkit Component Descriptions.md** | Project context file for AI assistants. Summarizes the project (architecture docs toolkit for game engine codebases via local Ollama LLM), directory layout, pipeline order, key `.env` configuration, clangd/LSP setup, and available presets.                                       |
| **Architecture Analysis Toolkit Ochestration.md**           | Usage guide for `Arch_Analysis_Pipeline.py` (the pipeline orchestrator) and its test suite. Covers CLI options (`--dry-run`, `--start-from`, `--skip-lsp`), directory structure, and troubleshooting.                                                                                       |
| **FileReference.md**                                        | Index of every file in the `LocalLLMAnalysis/` directory. Lists each pipeline script, prompt file, configuration file, and documentation file with a short description of its purpose and key functions.                                                                                  |
| **CLI Instructions reference.md**                           | Complete CLI reference for all pipeline scripts. Documents every parameter, flag, and option for `archgen_local.ps1`, `archxref.ps1`, `archgraph.ps1`, `arch_overview_local.ps1`, `archpass2_context.ps1`, `archpass2_local.ps1`, `serena_extract.ps1`, `generate_compile_commands.py`, and `llm_common.ps1`. Also covers `.env` variables, presets, and common workflows. |
| **New Codebase Instructions.md**                            | Short checklist for adapting the toolkit to a new codebase: read the pipeline instructions, update `.env`, and review scripts for any needed changes.                                                                                                                                |
| **Optimizations.md**                                        | Catalog of active context-window optimization strategies: trivial file skipping, source truncation, compressed LSP context, adaptive output budgets, simplified prompts, targeted Pass 2 context, chunked overview synthesis, SHA1 incremental processing, and preset regex overrides. Includes a per-request token budget breakdown. |
| **Architecture Analysis Toolkit - Quickstart.md**           | Condensed setup and usage reference. Covers `.env` configuration, the six-step pipeline with commands, optional LSP extraction, script option tables, preset list, output directory structure, and resumability.                                                                      |
| **Serena Toolkit Documentation.md**                         | Technical reference for the architecture pipeline. Covers project goals, environment details, pipeline stages, local LLM integration, clangd/LSP setup and extraction design, `compile_commands.json` generation, context optimization strategy, configuration files, and lessons learned. |
| **Architecture Analysis Toolkit - Setup & Usage Guide.md**  | Full setup and usage guide. Walks through prerequisites (Ollama, PowerShell, optional clangd/Python), installation, `.env` configuration, clangd/LSP setup with `compile_commands.json` generation, running the pipeline, presets, a worked example for the C&C Remastered Collection, output directory structure, resumability, and troubleshooting. |
